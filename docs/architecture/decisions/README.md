# Architecture Decision Records

Documents significant architectural decisions.

## Creating ADRs

Use `/dof:architect <topic>` or `/dof:decide` (captures from current conversation).

## Format

`ADR-NNN-title.md` where NNN is sequential (001, 002, ...).

## Statuses

- **proposed** — Under discussion
- **accepted** — Decision made, being implemented
- **deprecated** — No longer applies
- **superseded** — Replaced by another ADR
